\# Crypto Price Dashboard



A real-time cryptocurrency dashboard built using Python, Streamlit, and the CoinGecko API.



\## Features



\* Live Bitcoin Price

\* Live Ethereum Price

\* Live Solana Price

\* Live Dogecoin Price

\* INR Currency Support

\* Clean Dashboard Interface



\## Technologies Used



\* Python

\* Streamlit

\* Requests

\* CoinGecko API



\## Installation



```bash

pip install -r requirements.txt

streamlit run app.py

```
